# Threshold Training Pace App

Aplicação web estática baseada nas fórmulas da planilha “Jon's Threshold App Master 2.2”.

## Como usar
1. Abra `index.html` no navegador.
2. Escolha o tipo de cálculo.
3. Informe o tempo no formato `m:ss`, `mm:ss`, `352`, `3121` ou segundos.
4. Clique em **Calcular**.
5. Use **Adicionar à lista** para comparar atletas.
6. Use **Exportar CSV** para baixar a tabela calculada.

## Funcionalidades
- Cálculo do T-30 pace / 100.
- Testes de 200, 300, 400, 500, 2000 e 3000.
- Médias intervaladas de 200, 300 e 400.
- Ritmos por zona: White, Pink, Red, Blue e Purple.
- Frequência cardíaca de referência por zona.
- Ritmos para 50, 100, 150, 200, 300, 400 e 500.
- Lista local de atletas com armazenamento no navegador.
